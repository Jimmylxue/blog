# snow-tiny 

大家好我是jimmy，在我又为这周周末应该出什么视频而焦头烂额时，群里的一位小伙伴给了我一个非常棒的提示，他发现我的博客中的图片资源都太大了！

![](https://vitepress-source.oss-cn-beijing.aliyuncs.com/typoraimage-20221120170010923.png)

![](https://vitepress-source.oss-cn-beijing.aliyuncs.com/typoraimage-20221120170036226.png)

确实，之前一直没有注意这块的内容，一个小小的图标png图标，就高达600多k，这个是有点吃不消的，怪不得oss一直在扣费呢。大部分的流量其实都是被这些图标所吸走的。所以我决定，给这些图片瘦瘦身，压一压。也是为了我的钱包考虑，万一哪个视频火了，oss能少扣钱钱😄

说干就干，于是我综合了网上很多的文章、耗时近两天，终于完成了这个 **snow-tiny 自动化压缩工具（脚本）**

这里主要参考的是掘金上这个同学的文章，和代码，能这么快写好主要还是归功于这个同学：[掘金文档](https://juejin.cn/post/7153086294409609229)

因为我是一个前端程序员，所以在开发中我比较多想法都是顺着对于前端来说怎么样用着方便的思路写的，相信在看视频的前段小伙伴也会很喜欢的。
