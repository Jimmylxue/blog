# 奔跑吧，程序员

这本书是我在公司时，完成了当日的任务之后选择读的一本书，咋说呢，可以理解成技术书，但是又没有那么的技术，更多的是教我们如何“奔跑”，如何创业，如何成为一个更好的程序员。

第一部分的内容就是创业相关的内容，告诉了我们什么是**科技创业公司**。简单的说，创业创业公司的定义并不是看它值多少钱，有多少员工。最重要的一点是：**在极度不确定的条件下创造新产品或服务的人类组织**。

后面告诉我们为什么应该在创业公司工作：

- 获得更多的机会

  有机会能够决定和落地一些事情，在大公司我们只是一个螺丝钉。

- 更多的所有权

  初创团队员工。

- 更多的乐趣

  很多东西都是未知的，不是在舒适区，有更多的挑战。

为什么不应该在创业公司工作

- 更少的报酬，收入

- 更加艰苦的环境

  周末大家都在和家人享受生活时，我们得关心网站是否安稳运行。

  ......

- 创业就是牺牲

  对身体来说是一个非常大的考验。

- 可能也不会变的富有

  真正能成功的创业

见仁见智，主要是看大家怎么想和怎么选。
